# vaahprompts
AI Prompts to Build new projects


## How to use this Repository
Step 1: Add this repo as Sub Module in your project repo

Step 2: Run **vaahprompts/vh-project-requirements-gathering.md** prompt to generate project requirements

Step 3: Run **vaahprompts/vh-project-scope-prompt-generator.md** prompt to generate project scope AI prompt

Step 4: Run **vaahprompts/vh-project-planner-prompt-generator.md** prompt to generate project plan AI prompt
